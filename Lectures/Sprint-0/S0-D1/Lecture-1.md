## The structure of the web

- The client(***frontend***): The application the end user interacts with, think of amazon, ebay, tripleten's website
- The server(***Backend***): The application that provides information for client(frontend) side applications, it manages the database, creating, deleting and updating information.
- The Database: A tool that allows us to store and access business data

## **HTML**: Hyper-Text Markup Language

The backbone of the web

<!-- element class="fragment" -->

It's the language used to structure and display content on websites

<!-- element class="fragment" -->

It's based on tags `<html>`, tags are represented with `<` to open a tag and `>` to close a **tag** 

<!-- element class="fragment" -->

---
## Quick example
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <!-- 
    <p> is a paragraph tag 
  -->
  <p>Hello World</p>
</body>
</html>
```

---
## CSS: Cascading Style Sheet

- It's used to modify the way html is represented

<!-- element class="fragment" -->

- It sets design properties like font styles, dimensions(width/height), colors, spacing, etc

<!-- element class="fragment" -->

---
 ## Quick example

```css
p {
	/*
		Sets the font color
		to red on the paragraph tags
	*/
	color: red;
}
```
